# Aplikasi Web Portofolio Dinamis

Tugas Pengembangan Aplikasi (PA) - Sistem Portofolio Berbasis Web menggunakan Python Flask, TiDB Cloud, Cloudinary, dan Resend API.

## Fitur Aplikasi:
* Halaman Utama Portofolio Dinamis
* Dashboard Admin Panel (CRUD Data)
* Upload Gambar Manajemen Proyek via Cloudinary
* Pengiriman Notifikasi Kontak via Resend API
